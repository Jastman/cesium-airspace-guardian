NYC Airspace Guardian

SECTOR: MANHATTAN // LIVE FEED

System Overview

NYC Airspace Guardian is a real-time situational awareness dashboard designed for monitoring urban airspace traffic.

KEY CAPABILITIES

  • Live Tracking: Real-time integration with OpenSky Network API.
  • Threat Isolation: Automated geofence violation detection (Red Dome).
  • Tactical Visualization: 3D city integration using Google Photorealistic 3D Tiles.

DATA SOURCES

  • Terrain/City: Google Map Tiles API (Photorealistic 3D Tiles)
  • Live Flight Data: OpenSky Network API (REST)
  • Assets: Cesium Ion (SH-60B Seahawk)
